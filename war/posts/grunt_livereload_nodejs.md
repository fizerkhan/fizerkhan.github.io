Grunt is amazing task runner for front end application. We are using grunt for our front end project and it works great. But for a while, we don't know how to run grunt live reload with our own Node JS server.

The `connect` task in above code run its own server to serve the static pages.
And the `livereload` task refreshes the pages if any changes in the files that is mentioned in the `watch:livereload` task. Above code works well when you don't need/have your own server to serve pages.

Incase if you have your own server, then live reload must refresh the pages served from your own server. How can we do it?

### Solution

You don't need default static server. So go and remove `connect` task from Gruntfile

```javascript
connect: {
    options: {
        port: 9000,
        // change this to '0.0.0.0' to access the server from outside
        hostname: 'localhost'
    },
    livereload: {
        options: {
            middleware: function (connect) {
                return [
                    lrSnippet,
                    mountFolder(connect, '.tmp'),
                    mountFolder(connect, 'app')
                ];
            }
        }
    },
    test: {
        options: {
            middleware: function (connect) {
                return [
                    mountFolder(connect, '.tmp'),
                    mountFolder(connect, 'test')
                ];
            }
        }
    },
    dist: {
        options: {
            middleware: function (connect) {
                return [
                    mountFolder(connect, 'dist')
                ];
            }
        }
    }
},
```

Also remove `connect:livereload` and `open` from `server` task

```javascript
grunt.task.run([
    'clean:server',
    'concurrent:server',
    'livereload-start',
    'connect:livereload',      ==> Remove This line
    'open',                    ==> Remove This line
    'watch'
]);
```

Then add following scripts in your html files.

```html
<!-- livereload script -->
<script type="text/javascript">
    document.write('<script src="http://'
        + (location.host || 'localhost').split(':')[0]
        + ':35729/livereload.js?snipver=1" type="text/javascript"><\/script>')
</script>
```

Then run grunt server to watch and livereload

```
$ grunt server
```

Now run your server and see the beauty of page refresh whenever you make changes in the file.
